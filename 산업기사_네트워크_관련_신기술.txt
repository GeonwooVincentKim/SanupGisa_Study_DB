1. IoT ->  사물들을 인터넷으로 서로 연결
2. M2M (Machine to Machine, 사물 통신)
-> 1. 무선 통신을 이용한 기계와 기계 사이의 통신
-> 2. 무선으로 통합하여 상호 작용하는 통신

3. 모바일 컴퓨팅 -> 휴대형 기기로 이동하면서 네트워크에 접속하여 업무를 처리할 수 있는 환경
4. 클라우드 컴퓨팅 -> 컴퓨팅 자원은 중앙 컴퓨터, 단말기로 인터넷을 언제 어디서나 사용 가능 (가상화)
5. 그리드 컴퓨팅 -> 초고속 인터넷망으로 연결하여 공유, 고성능 컴퓨터

6. 모바일 클라우드 컴퓨팅 
-> 1. 클라우드 컴퓨팅 인프라 구성
-> 2. ICT 기술

7. 인터 클라우드 컴퓨팅
-> 1. 클라우드 서비스 연동
-> 2. 컴퓨텅 자원의 동적 할당 가능
-> 3. 클라우드 서비스나 자원을 연결하는 기술

8. 메시 네트워크
-> 1. 차세대 이동 통신, 홈네트워킹, 공공 안전 등 특수 목적을 위한 새로운 방식의 네트워크 기술
-> 2. 대규모 디바이스의 네트워크 생성에 최적화
-> 3. 기존 무선 랜의 한계 극복

9. 와이선 (Wi-SUN)
-> 스마트 그리드 (장거리 무선 통신 (IoT), 서비스를 위한 저전력 장거리 (LPWA : Low-Power Wide Area))
    = 1. 전기의 생산부터 소비까지 전 과정에 정보통신 기술 접목
    = 2. 에너지 효율성 높이는 지능형 전력망 시스템

10. NDN (Named Data Networking)
-> 1. 콘텐트 정보 & 라우터 기능으로 데이터 전송 수행
-> 2. NDN 은 클라이언트 서버가 기존의 IP 망을 대체할 새로운 인터넷 아키택처

11. NGN (Next Generation Network, 차세대 통신망)
-> 1. ITU-T 개발, 유선망 기반의 차세대 통신망
-> 2. 이동 사용자 & 유선망 목표, 완전한 이동성 (Full Mobility) 제공 목표

12. SDN (Software Defined Networking, 소프트웨어 정의 네트워킹) -> 네트워킹 가상화하여 제어 및 관리
13. NFC (Near Field Communication, 근거리 무선 통신) 
-> 1. 고주파 (HF) 를 이용한 근거리 무선 통신 기술
-> 2. 아주 가까운 거리에서 양방향 통신 기술 지원하는 RFID
    = 사물에 전자 태그를 부착하고 무선 통신을 이용하여 사물 정보 및 주변 정보 감지하는 센서 기술

14. UWB (Ultra WideBand, 초광대역)
-> 1. 무선 디지털 펄스
-> 2. 0.5m/W 저전력, 1km 거리까지 전송 가능 & 땅속이나 벽면 뒤로도 전송 가능

15. 피코넷 (PICONET) -> 블루투스 기술, UWB 통신 기술 사용)
16. WBAN (Wireless Area Network) 
-> 1. Wearable (웨어러블) 또는 Implant (몸에 심는) 형태의 센서나 기기를 무선 연결
-> 2. 개인 영역 네트워킹 기술

17. GIS (Geographic Information System, 지리 정보 시스템) -> 지리적인 자료 관리 컴퓨터 응용 시스템 & 위성 이용
18. USN (Ubiquitous Sensor Network, 유비쿼터스 센서 네트워크)
-> 1. 센서가 무선으로 정보 수집
-> 2. RFID, 사물의 인식 정보, 주변의 환경 정보 탐지하여 네트워크에 연결하여 정보 관리

19. SON (Self Organizing Network, 자동 구성 네트워크)
-> 1. 주변 상황에 맞추어 스스로 망을 구성하는 네트워크
-> 2. 통신망 커버리지 및 전송 용량 확장 (경제성), 망의 운영과 관리 효율성 증대

20. 애드 훅 네트워크 (Ad-Hoc Network)
-> 1. 재난 현장에서 모바일 호스트 (Mobile Host) 
-> 2. 망 구성 후, 단기간 사용 & 유선망 구성하기 어려울 때 사용

21. 네트워크 슬라이싱 (Network Slicing)
-> 1. 코어 네트워크 인프라 => 다수의 가상 네트워크 (고객 맞춤형 서비스 제공)
-> 2. 3GPP (3rd Generation Partnetship Project) & 글로벌 이동통신 표준화 단체 => 5G (IMT-2020)
    = 이동 통신 관련 국제 표준을 위해 각국의 통신 관련 기관이 참여하는 기술협력 기구

22. 저전력 블루투스 기술 (BLE; Bluetooth Low Energy) -> 2.4 Ghz 주파수 대역 사용, 대기 상태에서는 절전모드 유지 
23. 지능형 초연결망 -> 스마트 시티 & 스마트 스테이션 => 데이터 트래픽 효과적으로 수용 (과학기술정보통신부 주관 사업)
24. 파장 분할 다중화 (WDM, Wavelength Division Multiplexing)
-> 1. 광섬유, 복수의 신호 (통신 회선)
-> 2. 파장 다른 광선들끼리의 간섭 X

25. 소프트웨어 정의 데이터 센터 (SDDC, Software Defined Data Center)
-> 1. 모든 자원 가상화, 소프트 웨어 조작만으로 관리 및 제어되는 데이터 센터
-> 2. 소프트웨어로 정의
-> 3. 다양한 소프트웨어 정의 기술
    = 1. 컴퓨팅, 네트워킹, 스토리지 가상화하는 기술
    = 2. 하드웨어와 독립, 유연한 제어 => 비용 절감 가능, 운영 편리

26. 개방형 링크드 데이터 (LOD, Linked Open Data)
-> 1. Linked Data + Open Data => 누구나 사용할 수 있도록 웹상에 공개된 연계 데이터
-> 2. 개별 URI (인터넷 식별자) 로 식별, 링크 정보 부여 => 상호 연결된 웹을 지향하는 모형
