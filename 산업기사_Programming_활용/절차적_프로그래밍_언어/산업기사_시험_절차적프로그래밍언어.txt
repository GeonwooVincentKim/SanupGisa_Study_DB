절차적 프로그래밍 언어
-> 일련의 처리 절차를 정해진 문법에 따라 순서대로 기술하는 언어


절차적 프로그래밍 언어의 종류
- 1. C 
  = 1. 1972 년 벨 연구소에서 개발됨
  = 2. 고급 & 저급 프로그래밍 언어
  = 3. 이식성 좋음
  = 4. 시스템 프로그래밍 언어

- 2. ALGOL 
  = 1. 수치 계산, 
  = 2. 논리 연산을 위한 과학 기술 계산용 언어
  = 3. PASCAL, C언어 모체

- 3. COBOL
  = 1. 사무 처리용 언어
  = 2. 영어 문장 형식
  = 3. 4개의 DIVISION

- 4. FORTRAN 
  = 1. 과학 기술 계산용 언어
  = 2. 수학과 공학 분야의 공식이나 수식과 같은 형태로 프로그래밍 가능
