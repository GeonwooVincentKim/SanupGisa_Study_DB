스크립트 언어
-> 1. HTML 문서 안에 직접 프로그래밍 언어를 삽입하여 사용하는 언어
-> 2. 분류
   = 1. 서버용 스크립트 언어
     => 1. ASP, JSP, PHP, 파이썬
     => 2. 서버에서 해석되어 실행 후, 결과만 클라이언트로 보냄

   = 2. 클리이언트용 스크립트 언어
     => 1. JS, VBScript
     => 2. 클라이언트의 웹 브라우저에서 해석되어 실행 됨


스크립트 언어의 종류
-> 1. JS
   = 1. 웹 페이지의 동작, 클라이언트 언어
   = 2. 클래스 존재 X, 변수 선언 필요 없음
   = 3. 아이디, 비밀번호, 수량 등의 입력 사항 확인하기 위해 사용, 서버로 데이터 전송


-> 2. VBScript
   = 1. Microsoft 사에서 JS 에 대응하기 위해 제작한 언어
   = 2. ActiveX

-> 3. ASP
   = 1. 서버 측에서 동적으로 수행되는 페이지를 만들기 위한 언어
   = 2. Microsoft 사에서 제작
   = 3. Windows 계열

-> 4. JSP
   = 1. Java 로 만들어진 서버용 스크립트 언어
   = 2. 다양한 운영체제

-> 5. PHP
   = 1. Linux, Unix, Windows 사용 가능한 서버용 스크립트 언어
   = 2. C, Java 등과 문법 유사, 배우기 쉬움

-> 6. Python
   = 1. 객체지향 기능 지원, 대화형 인터프리터 언어
   = 2. 플랫폼에 독립적, 문법 간단

-> 7. Shell Script
   = 1. Unix / Linux 계열의 Shell 에서 사용되는 명령어 조합으로 구성된 Script 언어
   = 2. Shell Script 제어문 (if, case, for while, until)

-> 8. Basic
   = 1. 절차지향 기능 지원, 대화형 인터프리터 언어
   = 2. 초보자도 쉽게 사용 가능한 문법 구조
