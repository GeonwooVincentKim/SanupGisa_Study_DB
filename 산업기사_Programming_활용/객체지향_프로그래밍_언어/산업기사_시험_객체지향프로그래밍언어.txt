객체지향 프로그래밍 언어
-> 현실 세계의 개체를 하나의 객체로 만들어, S/W 를 개발할 때도 객체들을 조립해서 프로그램을 작성하는 방식


객체지향 프로그래밍 언어의 종류
-> 1. JAVA
   = 1. 분산 네트워크 환경
   = 2. 멀티스테드 기능, 여러 작업 동시 작업 가능
   = 3. 운영체제 및 하드웨어에 독립적, 이식성 강함

-> 2. C++
   = 1. C언어 + 객체지향 개념
   = 2. 모든 문제를 객체로 모델링하여 표현

-> 3. Smalltalk
   = 1. 1세대 객체지향 프로그래밍 언어, 순수한 객체지향 프로그래밍 언어
   = 2. 최초로 GUI 제공
